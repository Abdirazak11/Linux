<h2>Core Components of a Linux Machine</h2>

<pre>
+----------------------------------------------------+
| User Applications (Vim, Docker, Apache, etc.)      |
+----------------------------------------------------+
| Shell (Bash, Zsh, Fish, etc.)                      |  &lt;-- Part of the OS
+----------------------------------------------------+
| System Libraries (glibc, libc, OpenSSL, etc.)      |  &lt;-- Part of the OS
+----------------------------------------------------+
| System Utilities (ls, grep, systemctl, etc.)       |  &lt;-- Part of the OS
+----------------------------------------------------+
| Linux Kernel (Process, Memory, FS, Network)        |  &lt;-- Core of the OS
+----------------------------------------------------+
| Hardware (CPU, RAM, Disk, Network, Peripherals)    |
+----------------------------------------------------+
</pre>

<h3>(a) Hardware Layer</h3>
<p>
The hardware layer consists of physical components such as the CPU, memory, storage devices,
network interfaces, and peripheral devices. The Linux kernel directly interacts with this layer
to manage system resources and provide abstraction to higher-level components.
</p>
