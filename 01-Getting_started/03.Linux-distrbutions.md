<h2>Linux Distributions</h2>

<p>
Linux distributions (distros) are different versions of Linux that package the Linux kernel with
various software, system utilities, and package managers. Each distro is designed for different
use cases, such as personal computing, server management, or security.
</p>

<h3>Popular Linux Distributions</h3>

<ul>
  <li>
    <strong>Ubuntu</strong> – One of the most beginner-friendly distros, widely used for personal
    and server use. It has great community support.
  </li>
  <li>
    <strong>CentOS</strong> (discontinued, replaced by AlmaLinux / Rocky Linux) – Previously a
    popular choice for servers, based on Red Hat Enterprise Linux (RHEL).
  </li>
  <li>
    <strong>Debian</strong> – A very stable and reliable distro, often used as a base for other
    distros like Ubuntu.
  </li>
  <li>
    <strong>Fedora</strong> – A cutting-edge distro that introduces new features before they reach
    RHEL.
  </li>
  <li>
    <strong>Arch Linux</strong> – A lightweight, rolling-release distro for advanced users who
    prefer full customization.
  </li>
  <li>
    <strong>Kali Linux</strong> – Designed for cybersecurity and penetration testing.
  </li>
  <li>
    <strong>Alpine Linux</strong> – A lightweight, security-focused distro commonly used in
    containers.
  </li>
</ul>
